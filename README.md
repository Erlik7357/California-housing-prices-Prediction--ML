# California House Price Prediction

This project predicts California housing prices using Machine Learning.  
The goal is to estimate the **median house value** based on various features such as income, location, number of rooms, and population.

---

##  Project Objective

To build a regression model that can accurately predict house prices using structured housing data.

---

##  Dataset Information

The dataset contains the following features:

- `longitude`
- `latitude`
- `housing_median_age`
- `total_rooms`
- `total_bedrooms`
- `population`
- `households`
- `median_income`
- `ocean_proximity` (categorical feature)
